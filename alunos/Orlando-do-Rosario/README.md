Jornada nos Desafios da CBF Academy: 

Neste programa, trabalho em dois desafios progressivos e integrados:

Desafio 1 — Melhorar o SofaScore Dashboard
Partindo do projeto base em dashboard, copiei o código para minha pasta isolada e apliquei melhorias: novos filtros de análise, tratamento robusto de erros no frontend, sistema de cache com localStorage e lazy loading de imagens. O objetivo é deixar o painel mais resiliente, rápido e intuitivo para o usuário final.

Desafio 2 — Evolução Tática do Dashboard (StatsBomb)
Integro o motor analítico tático do statsbomb_aula ao meu dashboard, adicionando métricas avançadas como Expected Goals (xG), Expected Assists (xA), Pressão Pós-Perda (PPDA) e radares de performance de jogadores. O design deve seguir o padrão WOW Premium, com glassmorphism, dark mode elegante e responsividade total. Este desafio exige orquestração da equipe de agentes de IA para garantir qualidade arquitetural, pedagogia clara e validação visual.

Ambos os desafios trabalham a metodologia de Spec-Driven Development, onde atuo como Arquiteto de Software enquanto os agentes de IA implementam as especificações de forma automatizada e testada.
