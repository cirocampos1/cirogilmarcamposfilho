# Awesome NotebookLM Prompts

This skill provides access to a curated collection of the strongest NotebookLM & Kael.im slide prompts sourced from the real creative underground. These prompts are designed to turn papers, notes, transcripts, or random brain dumps into clean, structured, presentation-ready decks.

## Usage Instructions

When a user requests to generate slides, notes, or use NotebookLM with specific styles (e.g., "Modern Newspaper", "Manga Style", "Sharp-edged Minimalism"):

1. Read the local repository's `README.md` to access the exact prompt texts and design constraints:
   **Path:** `/home/leonardobarbosa/dev//tools/awesome-notebookLM-prompts/README.md`

2. You can use the `view_file` or `grep_search` tools to extract the specific prompt template from the `README.md` based on the user's requested style.

3. Combine the extracted prompt with the `notebooklm` CLI (via `notebooklm ask` or `notebooklm generate slide-deck`) or provide the prompt to the user to use in the NotebookLM UI.
